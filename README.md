# Data-Preprocessing-Assignment
This project predicts hotel booking cancellations using machine learning. It focuses on improving model performance through preprocessing, feature engineering, and feature selection processes.

# Project Overview
This project focuses on predicting whether a hotel booking will be canceled (is_canceled) using machine learning techniques.The main goal is to demonstrate how feature engineering, preprocessing, and feature selection improve model performance.

# Dataset
Source: https://raw.githubusercontent.com/swapnilsaurav/Dataset/refs/heads/master/hotel_bookings.csv

# Objective
The objective of this project is to build a model that predicts booking cancellations and analyze how preprocessing and feature engineering impact amd affect performance.

# Libraries and Tools used
Python, pandas, numpy, matplotlib, scikit-learn

# Tasks
Baseline model creation, Curse of dimensionality demonstration, Numeric preprocessing (binning, scaling, binarization), Distance-based learning (KNN) analysis, Pipeline creation using scikit-learn, Feature extraction from existing data, Feature construction (ratio, interaction, aggregation, polynomial features), Feature importance and feature selection

# Conclusion
- Model performance improved after preprocessing and feature engineering
- Scaling significantly improved KNN performance
- Feature selection reduced redundancy and improved efficiency
